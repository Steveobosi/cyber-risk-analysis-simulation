# 🔒 Cybersecurity Risk Analysis Simulation – GRC & Risk Management

## 📅 Duration
Mar 2026 – Present

## 📌 Project Overview
Completed a virtual cybersecurity risk analysis simulation focusing on Governance, Risk, and Compliance (GRC) practices.

This project demonstrates practical skills in conducting organizational risk assessments, evaluating cloud and third-party risks, and recommending effective security controls.

---

## 🛠️ Tools & Frameworks
- NIST SP 800-30 (Risk Management Guide)  
- AWS Cloud environment  
- Third-party vendor risk assessment techniques  

---

## 📂 Key Activities
- Conducted organizational risk assessments using NIST SP 800-30  
- Evaluated AWS environment and third-party vendor risks  
- Recommended security controls and compensating controls  
- Proposed mitigations, including Multi-Factor Authentication (MFA) and endpoint protection  

---

## 🎯 Skills Demonstrated
- Cybersecurity Risk Assessment  
- Governance, Risk & Compliance (GRC)  
- Cloud Security Evaluation (AWS)  
- Vendor Risk Management  
- Security Control Recommendations  

---

## 📊 Outcome
This simulation strengthened practical understanding of risk analysis and GRC processes in cybersecurity, preparing for real-world organizational security decision-making.
